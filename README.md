# Paradise Nursery

Paradise Nursery is a React-based e-commerce shopping cart application for an online plant shop.

## Project Name

**Paradise Nursery Shopping Application**

## Description

This project is a dynamic web application that allows users to:
- Browse houseplants organized by categories
- View plant details including images, names, and prices
- Add plants to a shopping cart
- Manage cart items (increase/decrease quantity, delete items)
- View total cart cost

## Technologies Used

- React
- Redux Toolkit
- Vite
- CSS3
